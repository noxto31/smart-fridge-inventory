# 家庭冰箱库存管理

管理家庭冰箱食材，追踪保质期，减少浪费。

## 当前第一版功能

- **手动添加食品**：输入食品名称后自动建议分类、存放位置和参考保存期限，用户可自由修改
- **小票导入**：上传小票图片（模拟识别），确认后批量导入食品
- **库存管理**：按冷藏/冷冻/常温分区查看，支持筛选和排序
- **临期提醒**：首页展示已过期、即将过期（2天内）和需关注（7天内）的食品
- **食品详情**：编辑食品信息，标记为已吃完或已丢弃
- **历史记录**：查看已处理食品的历史
- **分类规则库**：内置常见食品的分类和保质期参考规则

## 技术栈

- **框架**：Next.js 16 (App Router) + TypeScript
- **样式**：Tailwind CSS v4
- **数据存储**：浏览器 IndexedDB（Dexie.js）
- **表单管理**：react-hook-form + Zod
- **测试**：Vitest

## 本地启动方式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 运行测试
npm run test:run

# 类型检查
npm run typecheck

# 构建生产版本
npm run build
```

访问 http://localhost:3000 即可使用。

## 隐私说明

- 所有数据保存在浏览器本地 IndexedDB 中，不会上传到任何服务器
- 小票识别目前使用模拟数据，不涉及真实 API
- 请勿将真实家庭小票、个人数据或密钥提交到公开仓库
- 第一版不收集任何用户信息
