# 未来扩展路线图

## 优先级 1（近期待实现）

### 真实小票 OCR
- 接入 OCR API（如百度 OCR、腾讯 OCR）
- 替换 mockRecognizeItems 为真实 API 调用
- 支持多语言识别

### 数据导出与备份
- 导出为 JSON/CSV 文件
- 从文件导入恢复数据
- 定期提醒备份

### 消息提醒
- 浏览器通知：临期提醒
- 每日库存摘要

## 优先级 2（中期）

### 用户系统
- 手机号/邮箱注册登录
- JWT 会话管理

### 家庭共享空间
- 创建家庭组
- 邀请家庭成员
- 共享库存数据

### 云端同步
- 多设备数据同步
- 冲突解决策略
- 实时协作编辑

### 条形码扫描
- 扫描食品条形码
- 自动查询食品信息

## 优先级 3（远期）

### 智能推荐
- 基于库存推荐菜谱
- 食品搭配建议

### 自动采购
- 库存不足提醒
- 生成采购清单
- 对接电商平台

### 营养分析
- 食品营养成分查询
- 膳食均衡分析
- 消费统计报表

### 多平台
- 微信小程序
- iOS/Android 原生应用
