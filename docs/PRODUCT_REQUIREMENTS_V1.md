# 产品需求文档 V1

## 项目概述

家庭冰箱库存管理工具，帮助用户追踪冰箱中的食品库存和保质期，减少食品浪费。

## 目标用户

- 家庭主妇/主夫
- 注重食品管理的个人用户

## V1 功能范围

### 1. 库存首页 (/)
- 临期概览：已过期、2天内到期、7天内关注
- 存储区域卡片：冷藏、冷冻、常温
- 快捷操作入口

### 2. 手动添加食品 (/items/new)
- 表单录入：名称、数量、单位、购买日期、存放位置
- 自动建议：分类、存放位置、参考保质期
- 所有建议可手动修改

### 3. 小票导入 (/receipts/import)
- 图片上传（暂不接入真实 OCR）
- 模拟识别结果
- 可编辑后批量导入

### 4. 库存列表 (/inventory)
- 按存放位置筛选
- 按过期状态筛选
- 按分类筛选
- 默认按到期日排序

### 5. 食品详情 (/inventory/[id])
- 查看和编辑所有字段
- 标记已吃完/已丢弃
- 删除记录
- 转移存放位置

### 6. 历史记录 (/history)
- 已吃完食品列表
- 已丢弃食品列表

## 不做事项

- 用户系统（注册/登录/权限）
- 多人协作
- 云同步
- 真实 OCR API
- 条形码扫描
- 消息推送
- 菜谱推荐
- 自动采购
- 营养分析
- 商业化功能
